# Laravel-Task
An example app using [Laravel 5.3](https://laravel.com)

### UP & RUNNING
- Clone the repo
- `composer install`
- `cp .env.example .env`
- `php artisan key:generate`
- `Create a new MySQL database name - task`
- `php artisan migrate`
- `php artisan serve`
- Open your browser to `http://localhost:8000`
